Step up instruction for Mac

Step 1: put all the files in the same folder your create
Step 2: create a .env file and simply update the LLM API as simple text as shown below
  Deepseek_API_KEY = Enter your key here
  Google_API_KEY = enter your key here
Step 3: run the following commend in the terminal to sync up the packages used to run this app.
  Pip install -r requirement.txt
